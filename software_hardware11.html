Software and Hardware

Software Components

Operating System (OS)

Controls hardware and software, giving essential services for programs.

Application Software

Programs like word processors and media players that you use daily.

Middleware

Software that connects different applications, allowing them to communicate.

Firmware

Built-in software that manages hardware functions.

Hardware Components

CPU (Central Processing Unit)

The brain of the computer that processes instructions.

Registers

Small memory units inside the CPU that temporarily hold data.

ALU (Arithmetic and Logic Unit)

Handles math and logic operations inside the CPU.

Control Unit

Manages data flow and coordinates CPU actions.

Buses

Wires that transfer data between components.

RAM (Memory)

Temporary storage for data the CPU is using right now.

Storage Devices

Stores data permanently, like hard drives and SSDs.

GPU (Graphics Processing Unit)

Renders images and video for the display.

Power Supply

Converts electricity from the wall to power your computer.

Motherboard

The main board connecting all parts of the computer.
